# Инспектор входящих

Текстовый симулятор-свайпер о кибербезопасности. Игрок — стажёр отдела информационной безопасности компании «ТехноСфера» — за рабочую неделю (5 дней) разбирает входящую почту: фишинг отправляет в карантин, легитимные письма пропускает.

Полное техническое задание и сценарий — в [`CLAUDE.md`](./CLAUDE.md).

## Запуск

```bash
npm install
npm run dev
```

Откройте адрес, который выведет Vite (обычно `http://localhost:5173`).

## Сборка

```bash
npm run build
npm run preview   # локальный просмотр собранной версии
```

## Как играть

- **Свайп вправо** или кнопка **«Пропустить»** — письмо доставляется адресату.
- **Свайп влево** или кнопка **«В карантин»** — письмо блокируется как подозрительное.
- Две шкалы вверху экрана — **«Безопасность»** и **«Продуктивность»** — сквозные на всю неделю. Пропустил фишинг — падает безопасность; закарантинил легитимное письмо — падает продуктивность.
- Дойти до конца пятницы с обеими шкалами выше 70% — успех и зачисление в штат. Обнуление любой из шкал — немедленное поражение.

## Технологии

React + TypeScript + Vite, анимация карточек на `framer-motion`. Без бэкенда и внешних ассетов — всё самодостаточно.

## Структура

```
src/
├─ game/          # данные и чистая логика: карточки, дни, концовки, подсчёт очков
├─ components/     # презентационные компоненты экранов
├─ styles/         # стили корпоративного почтового клиента
├─ App.tsx         # состояние игры и переходы между экранами
└─ main.tsx
```
